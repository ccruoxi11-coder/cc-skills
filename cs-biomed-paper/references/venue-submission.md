# Venue submission mechanics

Submission logistics by venue family. Always confirm against the **current** author guidelines /
call-for-papers — these change yearly (principle: primary source first). Mark unknowns `[待补充]`.

## Universal pre-submission checklist

- Manuscript matches the target's **article type, word/figure limits, and template**.
- **Structured abstract** if required; **reporting-standard checklist/flow diagram** attached
  (CLAIM/TRIPOD+AI/CONSORT-AI/STARD/PRISMA as applicable).
- **Data & Code Availability** statements with real accessions/DOIs/license.
- **Ethics/IRB** statement, consent, conflicts, funding, author contributions (e.g. CRediT).
- Figures as vector / ≥300 dpi, de-identified; references in the journal's style.
- **Cover letter** stating the advance and fit; suggested/excluded reviewers if requested.

## Nature family (Nature subjournals, npj, Nat Med, Nat BME, Nat Comms)

- Often a **presubmission inquiry** for the flagship subjournals.
- **Reporting Summary** form (and Editorial Policy Checklist) required at submission.
- Strict main-text limits; extensive **Extended Data / Supplementary**; methods often at the end.
- Cover letter must argue significance to a broad audience and fit to the specific journal.
- Strong open-data/open-code expectations.

## IEEE TMI / Medical Image Analysis / TBME / JBHI

- Journal LaTeX/Word template; **double-column**; page/over-length charges may apply.
- Detailed Methods + thorough experiments expected; code release encouraged/required by some.
- ORCID, author bios (some IEEE journals), and graphical abstract sometimes required.

## MICCAI / CVPR / NeurIPS / ICLR

- **Hard deadline + page limit + official template**; supplementary allowed separately.
- **Double-blind** for many → anonymize text, acknowledgements, repo links (use anonymous repos).
- **Reproducibility checklist** often mandatory at submission; rebuttal phase is short and structured.
- MICCAI: still enforce sound medical-data handling and an ethics statement.

## General SCI / IF journal

- Read that journal's **Guide for Authors**: scope, article types, structured-abstract format,
  reference style, word/figure limits, data policy.
- Pick the journal *before* finalizing structure; align the cover letter to its aims & scope.

## Cover letter skeleton

`[待补充: editor name]` — We submit "[title]" for consideration as a [article type] in [journal]. This
work addresses [clinical/biological problem] by [approach], and shows [headline result with external
validation]. It fits [journal]'s scope because [reason]. The work is original, not under review
elsewhere, and all authors approve. Data/code availability: [statement]. We suggest reviewers: [names].
