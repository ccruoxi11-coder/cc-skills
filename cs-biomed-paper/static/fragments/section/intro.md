# Section: Introduction

## Default structure

`clinical/biological context & burden → what's currently done & its limits → methodological gap →
this work (approach + claim) → contributions list → (optional) roadmap`.

## Rules

- Open with the **clinical/biological stakes**, with a sourced fact (incidence, cost, diagnostic
  limitation) — cite it, don't assert it.
- Build the **dual gap**: clinical need *and* why existing methods fall short. End the gap exactly
  where your contribution begins.
- State contributions as concrete, verifiable bullets near the end.
- Funnel from broad to specific; every paragraph does one job (context / limits / gap / approach).

## Pitfalls

- A pure-CS motivation with no clinical reason to exist.
- Citing a statistic without a source, or an outdated burden figure → `[待补充：流行病学数据来源]`.
- Listing methods before establishing the gap.
- Over-claiming novelty the Discussion can't defend.
