# Section: Results

## Rules

- Report **observations only** — no interpretation (that's Discussion).
- Every headline number carries **uncertainty (95% CI) and an effect size**, with the comparator.
- Lead with the primary outcome, then secondary, then ablations, then subgroup/failure analysis.
- Report **internal then external** validation; show **calibration** for risk models, not only AUC.
- Numbers must match the abstract and the tables/figures exactly.
- State subgroup results and named failure modes honestly — don't report only the wins.

## Pitfalls

- Mixing in interpretation/claims ("this proves…").
- Bare point estimates; cherry-picked metrics; missing external/calibration results.
- p-values without effect sizes; uncorrected multiple comparisons.
- Inconsistent numbers across abstract/text/tables (top self-audit failure).
