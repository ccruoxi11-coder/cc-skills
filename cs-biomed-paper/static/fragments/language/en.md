# Language: English

Use when the user's notes are already in fluent English.

## Focus

- **Substance over surface.** Spend effort on argument structure, claim calibration, and
  evidence–claim proximity rather than basic grammar.
- **Hedging calibration** — match verbs to evidence (`show` / `suggest` / `may`); strike unsupported
  novelty (`first`, `unprecedented`) unless the evidence supports it.
- **Concision** — cut filler (`it is important to note`, `in order to`, `due to the fact that`);
  prefer active voice for the authors' actions.
- **Consistency** — one canonical term per concept; consistent tense (past for what was done, present
  for established facts); parallel structure in lists.
- **Flow** — one idea per paragraph, topic sentence first, explicit connectives between sentences.

## Output convention

Return the edit plus a short note on any change that touches **meaning or claim strength** (not minor
style), so the author can confirm intent. Never alter a scientific claim while editing language.
